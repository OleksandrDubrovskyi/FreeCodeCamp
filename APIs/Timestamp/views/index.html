Welcome to Viridian Crab

a Timestamp Microservice

Here's how to use it:

  1. The API endpoint is:
    https://viridian-crab.glitch.me/api/timestamp/your_date_string
  2. Your input must be a valid date string.
    The unix timestamp needs to be an integer (not a string) specifying milliseconds.
    We will use date strings compliant with ISO-8601 (e.g. "2016-11-20") because this will ensure an UTC timestamp.

Example Usage:

  • https://viridian-crab.glitch.me/api/timestamp/2015-12-25
  • https://viridian-crab.glitch.me/api/timestamp/1450137600

Example Output:

{"unix":1451001600000, "utc":"Fri, 25 Dec 2015 00:00:00 GMT"}
